# Scalable URL Shortener App


## Stack

This project uses the following stack:

- **React** (Frontend)
- **Node.js** (Backend)
- **Redis** (Cache & TTL Management)
- **MongoDB** (Database)
- **Apache ZooKeeper** (Token Generation)
- **Nginx** (Reverse Proxy & Load Balancer)
- **Docker** (Containerization)
